AI API

Base URL: https://ai-api.voidium.uk

Get Access

In Discord, run /aikey to get your personal API key (1 per account).

If you leave the Discord server, your key is deleted.

Auth & Limits

All API endpoints require:

Authorization: Bearer <your_api_key>

Rate limit: 10 requests / 4 minutes per key.

Endpoints

  • GET /v1/models — list available models.
  • POST /v1/chat/completions — OpenAI-compatible chat completions.
  • POST /api/chat — simple wrapper that accepts {"content":"..."} or {"messages":[...]}.

Example: /api/chat

curl -s https://ai-api.voidium.uk/api/chat \
  -H "Authorization: Bearer <your_api_key>" \
  -H "Content-Type: application/json" \
  -d '{"content":"Say hello","temperature":0}'

Example: /v1/chat/completions

curl -s https://ai-api.voidium.uk/v1/chat/completions \
  -H "Authorization: Bearer <your_api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "llama-3.1-8b-instant",
    "messages": [{"role":"user","content":"Reply with ok"}],
    "temperature": 0
  }'

Tip: set "stream": true and use curl -N for streaming.
